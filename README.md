# MIS
设计并实现一个民航票务管理系统。要求该系统具有票务管理(录入、修改、删除各航班机票信息)的功能和查询各航班信息的功能，其中票务管理功能只能由管理员进行。

需录入的信息如下：航班号、起点、终点、日期、起飞时刻、 到达时刻、票价、折扣票数、剩余座位数、航班所属航空公司。
